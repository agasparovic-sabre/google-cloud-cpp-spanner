Detailed Description

Represents the result of a read operation using spanner::Client::Read().

Note that a QueryResult returns the data for the operation, as a single-pass, input range returned by Rows().

Returns a RowParser which can be used to iterate the returned std::tuples.

Since there is a single result stream for each QueryResult instance, users should not use multiple RowParsers from the same QueryResult at the same time. Doing so is not thread safe, and may result in errors or data corruption.
