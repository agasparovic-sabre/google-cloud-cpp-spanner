Detailed Description

Represents the result of a query operation using spanner::Client::ExecuteQuery().

Note
ExecuteQueryResult returns both the data for the operation, as a single-pass, input range returned by Rows(), as well as the metadata for the results, query plan (if requested), and execution statistics (if requested).
