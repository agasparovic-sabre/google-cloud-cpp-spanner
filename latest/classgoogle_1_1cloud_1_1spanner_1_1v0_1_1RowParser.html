Detailed Description

template<typename TupleType>
class google::cloud::spanner::v0::RowParser< TupleType >

A RowParser converts the given ValueSource into a single-pass iterable range of std::tuple<Ts...> objects.

Instances of this class are typically obtained from the ResultSet::Rows<std::tuple<Ts...>> member function. Callers should iterate RowParser using a range-for loop as follows.

Warning
Moving a RowParser invalidates all iterators referring to the moved-from instance.
Example
ValueSource vs = ...
RowParser<std::tuple<bool, std::int64_t>> rp(std::move(vs));
for (auto row : rp) {
if (!row) {
// handle error
break;
}
bool b = std::get<0>(*row);
std::int64_t i = std::get<1>(*row);
// Using C++17 structured bindings
auto [b2, i2] = *row;
}
